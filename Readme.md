# 🤖 AI Flow Dashboard (MERN + OpenRouter)

A full-stack MERN application that allows users to input a prompt, generate AI responses using OpenRouter, visualize the flow using React Flow, and save results to MongoDB.

---

## 🚀 Features

- 🧠 AI-powered responses using OpenRouter API
- 🔄 Interactive flow visualization with React Flow
- 💾 Save prompts & responses to MongoDB
- ⚡ Fast frontend using React (Vite)
- 🎨 Responsive UI with Tailwind CSS
- 🔔 Toast notifications (no alerts)
- 📋 Copy response feature
- 🧹 Clear input/output

---

## 🛠 Tech Stack

### Frontend:

- React (Vite)
- Tailwind CSS
- React Flow
- Axios
- React Hot Toast

### Backend:

- Node.js
- Express.js
- MongoDB (Mongoose)
- OpenRouter API

---

### Clone Repository

```bash
git clone https://github.com/Ankitamishra17/mern-ai-flow-dashboard
cd mern-ai-flow

##  Setup Instructions

Backend Setup

cd backend
npm install

Run backend:
node server.js


Frontend Setup

cd frontend
npm install

Run frontend:
npm run

```
